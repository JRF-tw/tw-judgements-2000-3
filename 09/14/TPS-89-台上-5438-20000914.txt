【裁判字號】    89,台上,5438
【裁判日期】    890914
【裁判案由】    違反肅清煙毒條例
【裁判全文】
最高法院刑事判決                              八十九年度台上字第五四三八號
    上  訴  人  甲○○  男
    選任辯護人  舒建中律師
右上訴人因違反肅清煙毒條例案件，不服台灣高等法院中華民國八十九年六月十四日
第二審更審判決（八十九年度重上更(三)字第四十一號，起訴案號：台灣桃園地方法院
檢察署八十四年度偵字第一二五四○、一三四五四號），提起上訴，本院判決如左：
    主  文
上訴駁回。
    理  由
本件原判決認定：上訴人甲○○多次往返泰國，民國（下同）八十三年二月在泰國習
會施用毒品海洛因之惡習，乃於八十四年十月初，在泰國某處向「阿不拉」（真實姓
名不詳之台灣人）販入毒品海洛因後，意欲供己施用，而將之運輸來台，甲○○為分
散危險，乃以新台幣（下同）十五萬元之酬勞僱請泰國成年男子SAE SO SORAT（西元
一九五○年一月二十八日生，已據另案判處無期徒刑確定，現執行中），二人基於私
運如原判決附表編號一之管制物品及運輸如同附表編號一所示毒品海洛因來台之犯意
聯絡及行為分擔，由甲○○於八十四年十月九日，在泰國曼谷將其前所販入分別藏置
於休閒鞋墊下、內褲襠以及所攜帶之洗髮精一瓶、刮鬍膏一條等處之如同附表編號一
所示毒品海洛因等交予SAE SO SORAT，並提供SAE SO SORAT機票及安排 SAE SO SORA
T 來台後之旅館，指示SAE SO SORAT穿著該休閒鞋，帶同上揭夾藏毒品海洛因之內褲
、洗髮精及刮鬍膏等物，搭乘長榮航空公司ＢＲ二一二號班機於八十四年十月九日下
午四時五十分許入境桃園中正機場，SAE SO SORAT即遵照甲○○之指示搭機私運毒品
海洛因來台。甲○○另將其在泰國曼谷某飯店門口，同為供自己施用，以泰幣五仟元
（約新台幣五仟五佰元）所購買已將如同附表編號二之毒品海洛因夾藏於濾嘴與煙絲
中間之ＭＩＬＤ  ＳＥＶＥＮ香煙一包（香煙中亦混有同附表編號三所示不能分離之
海洛因），置入長褲褲袋內，搭乘上開長榮航空公司ＢＲ二一二同一班機同時入境中
正機場，而單獨私運上開部分之管制進口毒品海洛因來台。因SAE SO SORAT在機場久
候不見甲○○，乃先行投宿於與甲○○預先為其安排之台北市○○○路力霸大飯店六
二二室，以等候甲○○指定之人前來取貨。嗣因甲○○之友人沈榮因吸食毒品海洛因
（已據原審法院另案判處有期徒刑三年二月確定）於八十四年十月九日先搭乘泰國航
空公司ＴＧ六三四號班機於十二時二十分許返抵桃園中正機場，滯留於機場等候搭乘
另一班機入境之甲○○時，為法務部調查局台北市調查處調查人員當場查獲。沈榮供
出甲○○託泰國人SAE SO SORAT運輸毒品入境投宿於台北市飯店，調查人員因而循線
破獲SAE SO SORAT運輸毒品之犯行，於台北市力霸大飯店六二二室查獲SAE SO SORAT
，扣得夾藏於休閒鞋墊下、內褲褲前襠夾層、洗髮精、刮鬍膏中如同上附表編號一共
計淨重（驗餘）六百二十二點八八公克之毒品海洛因。甲○○則於八十四年十月九日
下午五時二十分許在桃園中正機場海關入境檢查室為警查獲，並在長褲之褲袋內查獲
兼供己施用摻有如同附表編號二所示海洛因及混有同附表編號三所示無法分離海洛因
成分之ＭＩＬＤ  ＳＥＶＥＮ香煙一包等情，係以上開犯行，業據上訴人坦承：在泰
國販入毒品海洛因，而由相識之泰國人SAE SO SORAT由泰國攜帶入境，且伊曾將此情
轉知友人沈榮，核與共同被告SAE SO SORAT在偵審中指陳：上訴人交付如原判決附表
編號一所示之海洛因指示伊運輸來台（見台灣台北地方法院檢察署八十四年偵字第二
二四四三號偵查卷第四十三頁；台灣台北地方法院另案八十四年重訴字第六十三號卷
第九、十、十一、一七二、一七三頁；訴緝字卷第一九四、一九五頁；原審法院另案
八十五年上重訴字第六十六號第十七、二七、二八頁）；證人沈榮證述：上訴人曾告
知有一位泰國朋友將運毒入境，故伊在機場等候上訴人，並確經伊供出上訴人與SAE
SO SORAT之聯絡號碼而查獲SAE SO SORAT與毒品（一審訴緝字卷第三○三、三○四頁
；原審上重更(二)卷第九五、九六頁）；證人施正聲證述：伊在泰國屢見上訴人與SAE
SO SORAT在一起，足見該兩人關係匪淺（一審訴緝字卷第二二二頁，原審上重更(二)卷
第一二五頁以下）各等情節相符，並有SAE SO SORAT被查獲書寫上訴人呼叫器號碼之
字據乙紙；運送毒品所穿之休閒鞋、內褲；所攜帶之洗髮精、刮鬍膏中之白色粉末
送驗結果確係海洛因之檢驗通知書三份；上訴人所夾帶藏有海洛因之香煙一包扣案佐
證，為其所憑之證據及認定之理由。而以SAE SO SORAT甫經查獲對上訴人清晰明確近
照之指認及供述較為可採，從而SAE SO SORAT嗣後翻異所供：上訴人不像伊所供「PE
TER 」其人云云，衡諸其對「PETER 」長相特徵描述始終如一，描述長相特徵復與上
訴人相符，足見上開SAE SO SORAT之翻供迴護上訴人，顯非可採。次以SAE SO SORAT
所供「PETER 」代伊購買機票乙節，縱其陳述如何購買之方式，先後不一，但代購者
確係「PETER 」其人，則始終不移，且「PETER 」即係上訴人已據SAE SO SORAT供明
，故上訴人聲請查詢伊有無以信用卡代購機票之紀錄，並無必要。再查，上訴人素有
吸食海洛因之惡習，且其涉嫌共同運輸毒品海洛因未達一公斤，與一般毒梟意圖營利
動輒運輸毒品數公斤以上者有別，是以上訴人辯其販入海洛因之目的，在供自己吸用
，尚非全無可能，應認不成立意圖販賣而販入毒品之罪。又查，上訴人運輸如原判決
附表編號一及編號二毒品（包括編號三摻有海洛因之香煙），係同時且分散為之，屬
整個單一運輸毒品之二個部分行為，為單純之一罪。公訴意旨僅就其中編號二所示毒
品（包含編號三所示部分毒品）起訴，其效力及於未起訴之上訴人與SAE SO SORAT共
同運輸同附表編號一所示毒品部分，應予一併審理，於理由內詳予指駁及說明。核上
訴人所為，係犯肅清煙毒條例第五條第一項運輸毒品之罪及懲治走私條例第二條第一
項私運管制物品進口二罪，前者，因毒品危害防制條例於行為後之八十七年五月二十
日公布，依刑法第二條第一項但書比較新舊法之結果，仍以適用上開舊法對上訴人有
利，又上訴人一行為同時觸犯上開二罪，為想像競合犯，應從一重之前罪處斷，且該
從一重處斷之前罪部分，上訴人與SAE SO SORAT互有犯意之聯絡與行為之分擔，均應
論以共同正犯。第以第一審關於上訴人運輸毒品部分，未及適用刑法第二條第一項為
上開新舊法之比較適用，且未將原判決附表編號二、三部分，僅上訴人單獨所犯，予
以區別等因，自屬無可維持，應予撤銷，爰適用肅清煙毒條例第五條第一項、第十二
條前段、第十三條第一項、懲治走私條例第二條第一項、第十一條、刑法第二條第一
項但書、第十一條前段、第二十八條、第五十五條前段、第三十七條第一項，酌情改
判論以上訴人共同運輸毒品，處無期徒刑，褫奪公權終身，扣案如原判決附表編號一
、二所示之毒品海洛因及摻有海洛因之MILD SEVEV香煙壹包均沒收銷燬之，內褲壹條
、休閒鞋壹雙、洗髮精壹瓶、刮鬍膏壹條均沒收，經核於法並無違誤。上訴意旨指陳
各項，經查，原判決就上訴人與SAE SO SORAT共同運輸毒品部分，已明係基於單純
一罪之關係及審判不可分之原則，併予審理之理由，並未變更檢察官起訴效力所及事
實以外之罪名，自無違背刑事訴訟法第九十五條第一款規定未告知變更罪名之可言。
次查，原審已明依憑共同被告SAE SO SORAT及證人沈榮、施正聲證詞，上訴人抗辯
：SAE SO SORAT運輸毒品係綽號「阿不拉」之指使云云，不符事實，並另無調查上訴
人有無以信用卡，代SAE SO SORAT購買機票之必要等理由綦詳，從而尚難以原審未調
查是否確有「阿不拉」其人，或未再傳提SAE SO SORAT、沈榮、施正聲查證SAE SOSO
 RAT是否受「阿不拉」僱用運輸毒品，遽指有違調查證據必要性之違誤。再查，原審
審判筆錄記錄提示內褲及休閒鞋（原審重上更(三)卷第九十四頁），固與原判決認定此
項物品業已銷燬不符；原判決理由中引用SAE SO SORAT警訊中所供年收入十五萬元，
與事實欄記載此十五萬元為其運輸毒品之代價，互不一致，惟均不足動搖原審依據其
他事證論定上訴人本案基本犯行之合法性，亦即不生影響判決之結果。其餘上訴意旨
，就原審採證認事之職權行使，徒憑己見，而為事實問題之爭執，核與法律規定得為
第三審上訴理由之違法情形，不相適合，尚難認其上訴為有理由，應予駁回。
據上論結，應依刑事訴訟法第三百九十六條第一項，判決如主文。
