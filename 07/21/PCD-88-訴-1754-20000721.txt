乙、被告方面：
一、聲明：駁回原告之訴及假執行之聲請；如受不利判決，願供擔保請免為假執行。
二、陳述：
（一）系爭土地原屬原告等與訴外人黃林阿于所共有，而原告等曾與黃林阿于成立分

（二）蓋原告等既出具同意書，且黃林阿于於六十九年間建築圍牆等建物後，使用系

      核無誤，並據以核發雜項執照，自不同能由他人所偽造。若該同意書為偽造，
      則黃林阿于建築圍牆時，原告等何不立即阻止，又豈容黃林阿于和平公然使用
      系爭土地長達十年餘。
（三）況原告乙○○○、丁○○及原地主林其祥三人之印章型式、字體等全然不同；
      其中林其祥部分更與印鑑章相符而為真正。而原告乙○○○之章為「葉桂蘭」

      於一般型式之章，顯見該印章並非他人所盜刻。再者於六、七十年間公寓房屋
      買賣之交易習慣，係法定空地或畸零地由建商於出售房屋時約定歸一樓使用，
      且原告等於嗣後向建商購屋時，業經建商約定而明知一樓法定空地或畸零地應

      十一日，始以買賣而登記取得系爭土地所有權，則於取得同意書時，並無須得
      丙○○之同意。
（四）又被告既本於分管協議而占有使用系爭土地，顯非無法律上之原因，而與不當
      得利之構成要件不合。又按善意占有人依推定其為適法所有之權利，得為占有
      物之使用收益。是占有人因此所獲得之利益，對於所有人不負返還義務，故原
      告主張被告應返還不當得利，於法顯有未合。遑論系爭土地位處巷道，非繁榮
      處，僅供被告自行堆置雜物，原告按申報地價百分之十計算租金，亦屬過高。
    理    由
一、原告起訴主張，系爭土地為原告與被告四人所分別共有，應有部分各四分之一。
    土地上除四人區分所有之四層建物一棟，占地面積為一一八點八平方公尺外，餘
    八二點八七平方公尺為法定空地。詎被告未經共有人全體之同意，擅於如附圖所
    示之土地上搭蓋地上物占有使用，侵害共有人權益，為此本於民法第七百六十七
    條規定請求被告拆除地上物及將占用土地返還全體共有人。又被告就本件法定空
    地全部占用，致原告受有不能使用之損失，而被告獲得相當於租金損害，為此本
    於不當得利法律關係，請求被告各給付原告每人相當於租金之損害等情。
二、被告則以，系爭土地原屬原告等與訴外人黃林阿于所共有，而原告等曾與黃林阿
    于成立分管協議，同意由黃林阿于建築圍牆等地上物，而使用系爭土地。是被告
    既向黃林阿于買受房屋及系爭土地、地上建物，被告自繼受前手黃林阿于之權利
    ，分管協議仍繼續存在，被告乃基於分管協議使用系爭土地，並非無權占。既非
    無權占有，則無原告所指不當得利，遑論該處空地並非繁榮，僅供被告自行堆置
    雜物，原告按申報地價百分之十計算租金，亦屬過高等語置辯。
三、原告主張系爭土地係屬原告與被告所共有，應有部分各四分之一，惟如附圖所示
    斜線部分（面積二五點五七平方公尺）占有使用中之事實，業據提出系爭土地登
    記簿謄本各一份為證，並經被告自認不諱，堪信為真實。惟被告另辯稱：系爭土
    地如附圖所示斜線部分，存有被告前手黃林阿于與斯時其餘土地共有人原告乙○
    ○○、丁○○及林其祥所締分管協議，黃林阿于並因此領有得建築圍牆之雜項執
    照。嗣黃林阿于將系爭土地應有部分四分之一轉售被告，林其祥將系爭土地應有
    部分四分之一轉售原告丙○○，然系爭分管協議對於彼等之後手應仍存在，其並
    非無權占有云云。
四、按「所有人對於無權占有或侵奪其所有物者，得請求返還之。」「共有物除契約
    另有訂定外，由共有人共同管理之。」「各共有人對於第三人，得就共有物之全
    部為本於所有權之請求。但回復共有物之請求，僅得為共有人全體利益為之。」
    民法第七百六十七條前段、第八百二十條第一項、八百二十一條定有明文，本件
    兩造所共有系爭土地如斜線所示部分遭被告占有使用之事實，為被告所自認，且
    經本院依聲請囑託台北縣新莊地政事務所派員測量，製有複丈成果圖附卷可參。
    則被告就其辯稱：系爭土地共有人就如附圖所示斜線部分土地存有分管契約，共
    有人同意被告就該部分土地為使用收益一節，既為原告所否認，被告自應就其所
    辯負舉證之責。雖被告聲請訊問證人黃林阿于；及聲請調閱台北縣新莊市公所六
    九雜字第三十三號雜項執照全卷（下簡稱雜項執照卷）、原告乙○○○、丁○○
    及訴外人林其祥之印鑑資料及系爭土地應有部分之買賣登記聲請書及契約書（公
    契）原本（下簡稱買賣契約卷），惟查：
（一）證人黃林阿于到庭結證稱：「伊原為系爭土地之共有人，伊買土地時建商有說
      可以使用畸零地，原告沒有告訴我（關於分管協議之事），在申請建造時，是
      由我公公去申請的，原告有蓋同意書同意我建造圍牆，以後就一直這樣用了，
      土地使用同意書是我公公去辦的。」等情；是依其證詞，關於系爭土地畸零地
      使用均由其公公處理，關於本院依聲請所調閱雜項執照卷內原告等所出具同意
      書，係由何人製作及蓋用印文，其均不知情，且未曾自其他共有人處獲悉同意
      使用事宜，僅於購買之初經建商告知得使用畸零地，是單憑證人前開證言，僅
      得證明系爭土地被告占用之圍牆為被告之前手黃林阿于所搭建，尚無得逕推論
      斯時之土地共有人原告乙○○○、丁○○及訴外人林其祥曾為分管使用同意。
（二）觀諸本院依聲請調取之雜項執照卷，依其內附之配置圖，申請之位置應含括本
      件被告占用之位置相符，此部分與證人黃林阿于之證詞相符，堪可採信，合先
      敘明。又卷內固附有申請時之共有人「乙○○○、丁○○及林其祥」之同意書
      。惟原告既否認其中「乙○○○」及「丁○○」印文之真正。被告自應就該部
      分事實為進一步之舉證。然經依被告聲請調閱原告乙○○○及丁○○於七十三
      年以前所留存之印鑑卡（印鑑證明申證書及委任書均罹十五年保存時效，業經
      銷燬，有台北縣新莊戶政事務所函一紙在卷可稽。），與同意書上之印文，以
      肉眼比對顯不相同，有印鑑卡三份附卷可佐。而以比對所調取買賣契約卷上「
      乙○○○」及「丁○○」印文，亦均與同意書之印文明顯不相符合（林其祥部
      分，則或十分相似。），是難憑前開資料，即可認同意書上原告「乙○○○」
      及「丁○○」印文為真正。
（三）被告雖另執：同意書上共有人三人之印章型式、字體等全然不同；而原告乙○
      ○○之章為「葉桂蘭」，與調解書上印章同樣使用「葉桂蘭」三字之印章，而
      原告丁○○之印章則小於一般型式之章，顯見該印章並非他人所盜刻；於六、
      七十年間公寓房屋買賣之交易習慣，係法定空地或畸零地由建商於出售房屋時
      約定歸一樓使用；因原告等及林其祥已出具同意書，故黃林阿于於六十九年間
      建築圍牆等建物後，使用系爭土地十餘年，原告等均未表示異議；況原告等在
      八十八年六月三十日聲請台北縣新莊市調解委員會調解時，亦表明其同意被告
      使用系爭土地，更足證確存分管契約云云。惟查，印文章之大小，使用印文之
      習慣，與同意書上印文是否真正，非有必然關係；交易習慣，或僅建商於售屋
      時之承諾，非必確得土地共有人之同意；原告於黃林阿于建築圍牆後，並未異
      議達十餘年，與默示成立分管契約尚屬有間。蓋所謂「默示之意思表示」須依
      原告之舉動或其他情事，足以間接推知其效果意思而言，原告單純沈默並無足
      為默示同意分管之推認；又原告三人於八十八年六月三十日向台北縣新莊市調
      解委員會聲請調解時，固陳明「前原一樓持有人因個人營生問題，曾徵得聲請
      人同意暫時使用建物所餘畸零地堆置營生器材：：」，惟同意「暫時堆置物品
      使用」與「長期搭建圍牆使用」亦有相當差距。
（四）因此，被告既未能提出其他事證證明其有何占有系爭土地如附圖所示斜線部分
      之權利，則原告主張被告係屬無權占有，堪信為真實。故原告基於所有權，請
      求被告將系爭土地上如附圖所示斜線部分地上物拆除，將土地返還原告等全體
      共有人，即有理由，應予准許。
五、另按「無法律上之原因而受利益，致他人受損害者，應返還其利益。」民法第一
    百七十九條前段定有明文。被告既無正當權源而占有使用兩造所共有系爭土地而
    獲利，致原告無法對系爭土地為使用收益，使原告受有損害，自屬不當得利。又
    無權占有他人土地，可能獲得相當於租金之利益為社會通常之觀念，是本件原告
    依不當得利之法律關係，主張被告無權占有使用系爭土地，為無法律上之原因而
    受利益，致原告受損害，應返還其相當於租金之利益，即非無據。惟原告雖依土
    地法第一百零五條準用同法第九十七條，以系爭土地之申報地價年息百分之十為
    其計算被告所獲相當於租金之利益之依據，而主張自八十三年十月一日起至八十
    六年六月三十日止，被告應給付原告每人（應有部分各四分之一）二萬零四十元
    （11400  x 25.57 x 0.1 x (2 + 9/12)/4 =20040）。自八十六年七月一日起至
    八十八年十二月三十一日止，應給付原告每人一萬九千一百七十八元（12000  x
    25.57  x 0.1  x (2  +  6/12) / 4 = 19178），合計共三萬九千二百十八元（
    20040  + 19178 = 39218）。另自八十九年一月一日起至拆除地上物交還土地之
    日止，每月給付每人損害金二千零七十二元（12000 x 25.57 x 0.1 x (1/12)/4
    =639）。經查：
（一）被告於八十三年四月十一日取得系爭土地所有權，有土地登記簿謄本一份在可
      憑。被告對於伊取得系爭土地後，即依現況使用一節，復未爭執，是原告請求
      被告自八十三年十月一日起至拆除地上物返還土地之日止，應給付相當於租金
      之利得，即屬有據。
（二）次查，系爭土地八十三年七月起之申報地價為每平方公尺一萬一千四百四十元
      、八十六年七月起之申報地價為每平方公尺一萬二千元，有地價謄本及土地登
      記簿謄本各一份在卷可憑，是原告主張八十三年十月一日至八十六年六月三十
      日止，按每平方公尺一萬一千四百元為計算基準，並未逾斯時申報地價，附此
      敘明。又查系爭土地為建地，面臨巷道，面積為二十五點五七平方公尺，目前
      供被告堆置雜物，並可供作停車用，有原告提出系爭土地地籍圖謄本及現場與
      附近相片二張在卷可稽，復為被告所不爭執。是本院審酌系爭土地上開坐落之
      位置、地目、被告占有利用系爭土地之經濟價值與所受利益等情狀，認原告請
      求以系爭土地申報地價總價額年息百分之十計算被告之不當得利，應屬過高，
      應以系爭土地上開申報地價依年息百分之四計算，方為允當。故依此計算，被
      告自八十三年十月一日起至八十六年六月三十日止應返還每位原告（原告計三
      人，每人應有部分均為四分之一）之不當得利為八千零十六元（11400      x
      25.57 x 0.04 x (2 + 9/12) x 1/4 = 8016）。八十六年七月一日起至八十八
      年十二月三十一日止，應給付原告每人七千六百七十一元（12000  x 25.57 x
      0.04 x (2 + 6/12) x 1/ 4 = 7671），合計共一萬五千六百八十七元（
      8016+ 7671 = 15687）。另自八十九年一月一日起至拆除地上物交還土地之
      日止，每月給付每人損害金二百五十六元（12000  x 25.57 x 0.04 x (1/12)
      x 1/4  = 256）。是原告請求被告返還不當得利，於上開範圍內為有理由，應
      予准許；逾上開部分之請求則無理由，應予駁回。
六、兩造陳明願供擔保，聲請宣告假執行或免為假執行，經核原告勝訴部分，合於法
    律規定，爰分別酌定相當之擔保金額宣告之；原告其餘假執行之聲請，因訴之駁
    回而失所依據，不予准許。
七、本件為判決之基礎，已足認定，兩造其餘攻擊防禦方法及舉證，核與本判決基礎
    無影響，無庸一一論列，附此敘明。
八、結論：本件原告之訴為一部有理由，一部無理由，依民事訴訟法第七十九條但書
    、第三百九十條第二項、第三百九十二條，判決如主文。
