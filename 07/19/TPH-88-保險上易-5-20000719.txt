    理    由
一、被上訴人起訴主張：訴外人台化公司自美國進口貨物一批，委由上訴人安邦公司
    運送，上訴人安邦公司則再委由中華航空股份有限公司（下稱華航公司）運送，
    詎前開貨物於抵台卸貨時至交付台化公司前全部遺失，致台化公司受有美金二萬
    七千五百八十元之損害，被上訴人為系爭貨物之保險人，已依約賠償台化公司前
    開損害折合為新台幣七十五萬七千八百九十八元，取得代位權，並受讓台化公司
    之權利，上訴人安邦公司為本件運送契約之當事人，依承攬運送契約、利益第三
    人契約，對台化公司負有損害賠償責任；爰基於保險代位及債權讓與之規定，依
    債務不履行之法則請求上訴人賠償美金二萬七千五百八十元或新台幣七十五萬七
    千八百九十八元，及法定遲延利息。
二、上訴人則以：(一)依被上訴人與台化公司間就系爭貨物之預約保險契約第六條第一
    、二、三項及類推適用海商法第一百七十四條之規定，被上訴人應提出台化公司
    為裝船之通知等事項，並向被上訴人陳報每次裝船之保險利益等具體資料之證據
    ，否則該保險契約失其效力；(二)台化公司僅為系爭貨物之受通知人，並非受貨人
    ，自非貨物所有權人，台化公司既未付款贖單，即不得主張提單上之權利；(三)被
    上訴人並未能證明業已理賠台化公司，況上訴人已於八十五年七月十五日與台化
    公司和解，台化公司自無任何權利可轉讓予被上訴人；(四)上訴人縱有賠償義務，
    因系爭貨物之託運人於交運貨物時並未就其在目的地交付時之價值作特別聲明並
    加付運費，故賠償責任亦應以每公斤二五○法郎相當於美金九．○七元為限等語
    ，資為抗辯。
三、查，被上訴人主張台化公司自美國進口貨物一批，委由上訴人運送，上訴人則將
    該批貨物交由華航公司運送之事實，業據提出空運提單為證，且為上訴人所不爭
    ，堪信為真實。而被上訴人主張系爭貨物業已遺失，伊為系爭貨物之保險人，已
    依約賠償貨主台化公司新台幣七十五萬七千八百九十八元，並受讓台化公司權利
    之事實，亦據提出貨物異常通知單、保險單、支出傳票、支票及代位求償書為證
    。上訴人雖以前開情詞置辯，惟查：
(一)、台化公司於八十四年一月一日與被上訴人簽訂海上貨物預約保險單，約定從世界
    各地的港口機場到台灣的任何港口機場，然後至被保險人的倉庫，若有任何索賠
    發生，均由保險人直接支付之事實，有被上訴人提出且為上訴人所不爭之保單及
    譯文附原審卷足憑。按諸現今國際貿易實務，交易極為頻繁，如須於每次交易時
    個別訂立保險契約，則必須由要保人與保險公司逐次議定保險費率、保險範圍、
    保險條款，如此必將不利於國際貿易之迅速要求。故在保險實務上，為節省每次
    訂定保險契約逐批貨載個別保險之麻煩手續，保險公司與貨主常以訂定預約保險
    方式，持續性地長期投保貨主之貨載。本件被上訴人與台化公司間之預約保險單
    條款（見原審卷第一三七、一三八頁）第六條第一項、第二項固規定：「被保險
    人應無例外地將每次裝船之保險利益（包括對貨物之性質、數量、包裝等）、保
    險標的物價值、航程、裝運船舶或航空器之名稱及起航日等事項，向保險人為明
    確陳報。：：若上揭事項在承保危險開始存在之時，尚未能得知者，被保險人應
    在得知上揭事項資料前為暫時之陳報，並在得知實際資料後為確定之陳報」；第
    三項亦規定：「對於前揭陳報，保險人應開出保險單或批示，以備保險人於保險
    利益受損時，據以向保險人請求賠償。」等語。惟此項約定被保險人有通知之義
    務（Declaration Clause），若被保險人未為通知，依同條第二項規定：「
    Failure to so declare shall at thiscompany's option render this open
    policy  null and void as from the date of such failure」（未依第一項為
    通知者，保險公司可選擇使本件預約保險契約自違反通知義務之日起失效），賦
    與保險公司有決定保險契約是否失效之裁量權，保險契約並非當然無效。且此約
    定亦無違保險法上之強制規定，自屬有效。至同條第三項規定：「被保險人為通
    知後，保險公司應出具保險單或批示，以便被保險人得據以就保險利益遭受損害
    一事申請理賠」（：：：which is necessavy for the Assured to make a
    claim against this Company for any loss or damage happening to the
    insured interest），則係便於被保險人辦理理賠證明之用，被保險人若另有其
    他申請理賠文件足供辦理者自無不可，是本件保險契約為有效至明，上訴人以被
    保險人違反「裝船通知義務」質疑本件保險契約之效力，殊屬無據。
(二)、次按運送提單，不論海運提單或空運提單，均為有價證券，而國際貿易通常以信
    用狀作為付款之方式，開狀銀行為擔保進口商能付款贖單，一般均以開狀銀行為
    受貨人，另以真正進口商為受通知人，俟貨物運抵目的地後，再由運送人通知進
    口商前往開狀銀行付款贖單，以憑領取貨物。本件提單上記載之受貨人為加拿大
    帝國商業銀行台北分行（下稱加拿大銀行），受通知人為台化公司，即台化公司
    為系爭貨物之進口商，加拿大銀行為開狀銀行。台化公司於八十五年一月三十日
    付款贖單，有加拿大銀行出具且為上訴人所不爭之確認書可參，則上訴人辯稱台
    化公司尚未付款贖單，並未取得提單上之權利云云，自不足採。又本件被上訴人
    係基於運送契約之法律關係對於上訴人請求貨物滅失之損害，其所提之空運提單
    ，僅作為契約憑證之用，是則，空運提單是否具有物權證券之性質，尚無庸深論
    。
(三)、又，被上訴人係開立八十五年七月十三日期，票號ＤＡ0000000面額為新
    台幣三，三０六，四八五元（尚包括其他應給付予台化公司之保險金）之支票支
    付本件保險金額，支出傳票中「賠案編號９６Ｍ１９３，保單編號０４００－６
    Ｐ0000000，理賠額七五七，八九八元」之一筆即為本件所支付之保險金
    ，保險代位求償書亦有相同之記載（見本院卷第六九至七五頁），該支票業經台
    化公司委託彰化商業銀行民生分行於同年七月二十二日兌現，則台化公司對上訴
    人之損害賠償請求權已當然移轉予被上訴人，被上訴人自得主張法定代位權。至
    上訴人固曾片面寄交發票日為八十五年七月十五日，面額新台幣六千四百六十元
    之支票予台化公司，惟台化公司早在此前即受領被上訴人理賠之支票，本無權接
    受上訴人和解，故於提兌該支票後，另行開立支票退回上開款項，有存證信函附
    原審卷第一七一頁可稽，足見台化公司並未接受上訴人以不及貨物價值十分之一
    即新台幣六千四百六十元賠償之合意，該公司縱曾兌現該支票，應係內部作業錯
    誤所致，尚不得據以認定台化公司已與上訴人達成和解。上訴人抗辯與台化公司
    達成和解，顯非可採。
(四)、再者，被上訴人係代位行使台化公司之債權，依涉外民事法律適用法第六、七條
    規定，應適用美國法為其準據法；而美國為華沙公約簽訂國之一，兩造對於本件
    契約關係，應適用華沙公約之規定，均無異議，是上訴人能否主張責任限制，胥
    依華沙公約之規定，不得任由上訴人藉其製訂之定型化提單條款予以變更。按華
    沙公約第二十二條第二項規定：「在貨物及經登記之行李之運送，除託運人在將
    貨件交與運送人時，有就貨物價值作特別聲明，並於必要時已加付費用者外，運
    送人之賠償責任以每公斤二五○法郎為限。」亦即運送人對於承運貨物之滅失、
    受損或延誤應負之賠償責任，以每公斤貨物二五○金法郎為限，除非託運人事先
    申報更高之價值，並依運送人之規定，加付運費。故運送人之單位責任限制，以
    貨物未經託運人聲明價值，並於被要求時加付運費者，始得主張。倘貨物已經託
    運人聲明價值，並依運送人之要求而給付費用者，運送人即不得主張單位責任限
    制。被上訴人持以請求之00  00000000號提單，既已記載託運貨物之價值為美金
    二萬七千五百八十元，縱與Declared Value for Carriage欄所載N.V.D不符，惟
    提單係由上訴人自行填載，非託運人所能知悉，且如何收取加付費用及金額多寡
    ，亦係由運送人決定，託運人或受貨人係依運送人之通知繳費，不能因提單未載
    明加值費用，即謂本件未申報價值，而影響託運人之權益，故上訴人主張單位責
    任，已屬無據。上訴人雖復辯稱：依本件提單背面約款所載，「報關申報價值」
    與「運送申報價值」並不相同，台化公司僅「報關申報價值」而未就「運送申報
    價值」作特別聲明，上訴人自得主張限制責任云云。然查，託運人向運送人就貨
    物價值做特別聲明後，運送人要將之記載在提單何處，並非託運人所能知悉或控
    制，上訴人雖在提單上之「託運人申報價值欄」（Sender's Declaration Value
    ）內再分兩小欄之「Declaration Value for Carriage」即運送申報價值，以及
    「Declaration Value for Custom」即報關申報價值，惟在託運人交運貨物就貨
    物價值為聲明時並無此項區別，上訴人以在提單背面定型化契約條款作為其所應
    負責任之範圍，主張填在「Declaration  Value    for Carriage」才不能享責
    任限制，填在「Declaration Value for Custom」則仍能享責任限制，與華沙公
    約之規定不符，殊不足為取。至上訴人所指本件買賣之貿易條件為ＦＣＡ，故運
    費由買方即受貨人台化公司負擔，伊有將年度運費費率表傳送予台化公司云云，
    與本件爭點全然無涉；所提之服務手冊有「報關申報價值」與「運送申報價值」
    概念不同，亦僅係上訴人公司內部人員參考之文件，不足憑為上訴人有利之證明
    。
四、綜上足見，上訴人之抗辯均非可採。被上訴人業於八十五年七月十三日已將系爭
    貨物價格美金二萬七千五百八十元折算為新台幣七十五萬七千八百九十八元理賠
    予台化公司，是則，被上訴人基於保險法第五十三條規定及債務不履行損害賠償
    請求權，請求上訴人給付新台幣七十五萬七千八百九十八元及自本件起訴狀繕本
    送達之翌日即民國八十七年十二月三十一日起至清償日止，按年息百分之五計算
    之利息，為有理由，應予准許。至被上訴人另請求上訴人或以美金二萬七千五百
    八十元支付部分，則因被上訴人於給付台化公司時，業已折付新台幣，其給付之
    內容及金額均屬確定，自無再請求給付美金之理，此部分之請求則應與其假執行
    之聲請併予駁回。原審就超過上開應准許部分，為上訴人敗訴之判決並為假執行
    之宣告，自有未洽。上訴意旨就此部分指摘原判決不當，求予廢棄改判，為有理
    由。至於上開應准許部分，原審判命上訴人給付，並為假執行之宣告，核無違誤
    ，上訴意旨，就此部分，仍執陳詞，指摘原判決不當，求予廢棄，為無理由，應
    駁回其上訴。
五、據上論結，本件上訴為一部有理由，一部無理由，依民事訴訟法第四百五十條、
    第四百四十九條第一項、第七十九條但書，判決如主文。
